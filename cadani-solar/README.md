# cadani-solar

Astro-Grundstruktur fuer eine performante, mobile-first Firmenwebseite im Solarbereich.

## Entwicklung

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` startet die lokale Entwicklung.
- `npm run build` prueft TypeScript/Astro und erzeugt den statischen Build.
- `npm run preview` zeigt den Produktionsbuild lokal an.
- `npm run check` fuehrt Astro-Checks aus.
